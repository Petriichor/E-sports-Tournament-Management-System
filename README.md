# E-sports Tournament Management System

A robust command-line application built with **Go (Golang)** designed to manage e-sports tournaments efficiently. This project was developed as the final assignment (Tugas Besar) for the Algorithm and Programming course.

## 📌 Project Overview
This application simplifies the process of organizing e-sports competitions, from team registration to match scheduling and winner determination. It focuses on implementing efficient algorithms and clean code structures in Go.

## ✨ Key Features
- **Team Management:** Create, update, and delete team profiles.
- **Tournament Bracketing:** Automated or manual match scheduling.
- **Score Tracking:** Real-time update of match results and standings.
- **Search & Filter:** Find specific teams or matches using optimized search algorithms.
- **Data Persistence:** Save and load tournament data using file handling.

## 🛠️ Tech Stack
- **Language:** Go (Golang)
- **Concepts:** Structs, Slices, File I/O, Sorting & Searching Algorithms.
